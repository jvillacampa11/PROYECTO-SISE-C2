Proyecto Final: Creación de una Página Web

El alumno deberá desarrollar una página web con un **tema libre**, aplicando los conocimientos adquiridos en el curso. Se evaluará el diseño, funcionalidad y usabilidad.

Requisitos:

  • Debe contener al menos 3 secciones (Inicio, Contacto, Información).
  • Uso de HTML, CSS y JavaScript básico.
  • Diseño responsivo para diferentes dispositivos.
  • Publicación en un servidor o uso de GitHub Pages.
